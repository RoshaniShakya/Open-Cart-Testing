Overview
This repository contains a complete Manual Testing Project for the OpenCart E-Commerce application. The project demonstrates end-to-end Software Testing Life Cycle (STLC) activities, including test planning, test design, execution, defect management, traceability, and reporting.

Project Scope: 
The testing effort focused on validating key OpenCart functionalities such as:
User Registration
Login & Authentication
Product Search
Product Comparison
Shopping Cart
Checkout Process
Order Management
Gift Voucher
Contact Us
Account Management
